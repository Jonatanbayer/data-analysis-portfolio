# Exploratory Data Analysis (EDA)

This category contains projects focused on **Exploratory Data Analysis (EDA)**,
with the goal of understanding data structure, identifying patterns,
trends, and potential anomalies.

The projects in this section demonstrate my ability to:
- Understand unfamiliar datasets
- Formulate questions based on data
- Explore distributions, correlations, and relevant variables
- Communicate findings through clear visualizations

## Key Steps Performed
- Data loading and initial inspection
- Descriptive statistical analysis
- Data visualization
- Identification of patterns and initial insights

## Technologies Used
- Python
- pandas
- numpy
- matplotlib
- seaborn

## Category Objective
To demonstrate proficiency in data exploration as a foundation
for deeper analysis and data-driven decision-making.
