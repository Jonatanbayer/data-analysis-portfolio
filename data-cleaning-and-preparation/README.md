# Data Cleaning & Preparation

This category is dedicated to projects where the primary focus is
**cleaning, organizing, and preparing data** for analysis.

The projects address common real-world data challenges,
such as missing values, inconsistencies, and unstructured data.

## Main Activities
- Handling missing values
- Data standardization and formatting
- Fixing inconsistencies
- Data transformation and structuring
- Preparing datasets for further analysis

## Technologies Used
- Python
- pandas
- numpy
